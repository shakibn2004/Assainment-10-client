# BloodBridge

> Connecting blood donors with those in need — saving lives across Bangladesh.

##  Live URL

[https://blood-hope.netlify.app](https://blood-hope.netlify.app)

---

##  Purpose

BloodBridge is a blood donation platform designed to bridge the gap between voluntary blood donors and recipients in urgent need. The platform allows users to register as donors, browse active blood requests, and connect with the right person at the right time — making the donation process faster, easier, and more accessible for everyone in Bangladesh.

---

##  Key Features

- **Donor Registration** — Users can sign up as blood donors with their blood group, location, and contact details.
- **Blood Request Listings** — Recipients can post blood requests, which are displayed in a searchable and filterable card grid.
- **Search & Filter** — Filter donation requests by blood group, district, or urgency level.
- **User Authentication** — Secure sign-up and login system for both donors and recipients.
- **Responsive Design** — Fully mobile-friendly UI built with a dark crimson theme for a bold, mission-driven look.
- **Animated UI** — Smooth scroll-triggered animations and transitions using Framer Motion.
- **Contact / Request Form** — A dedicated form for users to submit new blood requests or get in touch.
- **Real-time Donor Availability** — Donor profiles indicate availability status so recipients know who to reach out to.
